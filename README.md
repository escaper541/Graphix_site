# Tevez Multimedia

A multi-page static website for **Tevez Multimedia**, a creative services agency based in Tamale, Ghana. The site showcases their services, portfolio, and provides a way for clients to get in touch or place orders.

## Pages

- `index.html` — Homepage with hero, services overview, portfolio preview, testimonials, and FAQ
- `services.html` — Full services listing with order modals and Paystack payment integration
- `portfolio.html` — Filterable portfolio gallery with lightbox viewer
- `contact.html` — Contact form (via Formspree) and business info

## Services Offered

- Website Creation & Redesign
- Travel & Event Ticket Booking
- Premium Logo & Brand Identity
- Custom Digital Invitations
- Professional Document Design
- Photo Editing & Retouching
- Social Media Content Creation
- Custom Merchandise Design
- Event Photo Manipulations
- Mockup Creations
- Digital Art & Illustration
- Video Editing & Motion Graphics

## Tech Stack

- Plain HTML, CSS, JavaScript (no frameworks)
- [Font Awesome 6.4](https://fontawesome.com/) for icons
- [Google Fonts](https://fonts.google.com/) — Poppins & Orbitron
- [Formspree](https://formspree.io/) for contact form submissions
- Paystack for payment processing on service orders
- Custom canvas-based particle animation background

## Getting Started

No build tools or dependencies to install. Just open any `.html` file in a browser.

```bash
# Clone the repo
git clone <your-repo-url>

# Open in browser
start index.html
```
